# Telegram Content Automation Bot

A Python-based Telegram automation bot that fetches and sends random content such as jokes, quotes, anime quotes, and interactive prompts directly to a Telegram channel.

This project was created to explore:
- Telegram bot automation
- API integration
- Async programming in Python
- Content fetching and delivery systems

The bot automatically collects content from multiple public APIs and posts it in a formatted way.

## Features

- Sends random Hindi jokes
- Sends motivational quotes
- Anime quote generator
- "Would You Rather" prompts
- Image sending support
- Telegram channel automation

## Technologies Used

- Python
- Telegram Bot API
- asyncio
- requests

## APIs Used

- Hindi Quotes API
- Hindi Jokes API
- Anime Quotes API
- Would You Rather API

## How It Works

The script:
1. Fetches random content from APIs
2. Formats the response
3. Sends messages and images to Telegram automatically
